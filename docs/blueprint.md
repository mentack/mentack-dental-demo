# **App Name**: Mentack Dental

## Core Features:

- Multi-page Structure: Navigation and routing for Home, Treatments, Doctors, About, FAQ, and Contact pages.
- Bilingual Support: Dual-language support (TR/EN) with a language toggle. Strings should be managed in a translation dictionary. The language preference will be stored in local storage.
- Dynamic SEO: SEO optimization for each page, including dynamic titles, meta descriptions, OpenGraph tags, and schema.org markup.
- Appointment Request Form: Contact page appointment request form that emails the request or shows a confirmation and offers WhatsApp/call options.
- Treatment Pages: Individual treatment pages with structured data: title, short description, and sections detailing what it is, who it's for, how it works, and aftercare. Implement treatment-specific CTAs.
- Doctor Profiles: Doctor profiles that show the doctors names, job titles, short bio, etc.

## Style Guidelines:

- Background color: Near-white (#F8FAFC) to provide a clean and open feel.
- Primary color: Calm teal (#4DB6AC) to instill a sense of trust and serenity.
- Neutral text: Dark slate (#37474F) to ensure optimal readability.
- Accent color: Soft orange (#FF8A65) for CTAs to guide user action without overwhelming the design.
- Headings: 'Poppins' (sans-serif) at 600–700 weight to offer a contemporary and fashionable look while ensuring clear visual hierarchy. Note: currently only Google Fonts are supported.
- Body: 'PT Sans' (sans-serif) at 400–500 weight to ensure readability with a modern, humanist touch for longer texts. Note: currently only Google Fonts are supported.
- Simple line icons to maintain a clean and professional aesthetic.
- Max-width container: 1120px to maintain content structure and readability.
- Consistent section padding: 64–96px (desktop), 40–64px (mobile) to establish visual rhythm.
- Subtle hover effects and simple fade/slide-in animations to add visual interest without distracting from the core content.
